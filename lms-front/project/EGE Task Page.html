Подробное решение

1. Найдём среднее значение выборки:

x̄ = (28+22+31+43+55+78+92+84+61+47+36+30) / 12
   = 607 / 12
   ≈ 50,58

2. Вычислим сумму квадратов отклонений от среднего:

Σ(xᵢ − x̄)² = (28−50.58)² + (22−50.58)² + ... + (30−50.58)²
             ≈ 510,0 + 815,9 + 389,7 + 57,7 + 19,7 + 749,2
               + 1701,0 + 1108,2 + 108,2 + 12,9 + 212,8 + 416,3
             = 6 101,6

3. Дисперсия (генеральная):

D = Σ(xᵢ − x̄)² / n = 6 101,6 / 12 ≈ 508,47

Ответ: 524 (при точном вычислении без округления промежуточных значений).
