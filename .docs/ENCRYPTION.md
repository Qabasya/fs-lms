# ENCRYPTION.md — Настройка шифрования персональных данных (PII)

Плагин FS LMS использует алгоритм `XSalsa20-Poly1305` (через расширение `libsodium`, встроенное в PHP 7.2+) для шифрования чувствительных персональных данных на уровне приложения. Для корректной работы плагину требуются криптографические ключи, которые должны быть заданы в файле конфигурации WordPress.

## 1. Добавление констант в `wp-config.php`

Откройте корневой файл `wp-config.php` вашего WordPress и добавьте следующие строки **до** строки 

`/* That's all, stop editing! Happy publishing. */`:

```php
// Ключ шифрования PII (32 байта в бинарном виде, закодированные в base64)
define('FS_LMS_ENC_KEY', '<вставьте_сгенерированный_ключ>');

// Соль для хеширования (произвольная непустая строка)
define('FS_LMS_HASH_SALT', '<вставьте_случайную_строку>');
```

## 2. Генерация ключа шифрования
Ключ `FS_LMS_ENC_KEY` должен быть ровно 32 байта (256 бит) в бинарном виде, закодированный в base64. 
Для его генерации выполните в терминале следующую команду:
```bash
php -r "echo base64_encode(sodium_crypto_secretbox_keygen());"
```
Скопируйте результат выполнения команды и вставьте его в `wp-config.php` вместо `<вставьте_сгенерированный_ключ>`.
Для `FS_LMS_HASH_SALT` подойдёт любая уникальная строка. 
Рекомендуется использовать генератор случайных паролей или выполнить:
```bash
php -r "echo bin2hex(random_bytes(32));"
```

## 3. Требования к конфигурации

| Параметр | Требование | 
|----------|------------|
|FS_LMS_ENC_KEY |Ровно 32 байта в декодированном виде (обычно 44 символа в base64). Плагин не запустится, если длина не соответствует `SODIUM_CRYPTO_SECRETBOX_KEYBYTES`. |
|FS_LMS_HASH_SALT |Любая непустая строка. Должна быть уникальной для каждого окружения. |
|Хранение |Ключи находятся только в wp-config.php. Они никогда не сохраняются в БД, в коде плагина или в опциях настроек. |


## 4. Проверка конфигурации
После добавления констант активируйте плагин FS LMS. При наличии валидных ключей активация пройдёт успешно. Если ключи отсутствуют или имеют неверный формат, плагин автоматически деактивируется и выведет уведомление с инструкцией по настройке.
